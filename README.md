# rthz-CMC

A clean, responsive cryptocurrency tracking application built with React, TypeScript, and Tailwind CSS. Features real-time price data from CoinGecko API with a beautiful dark/light mode theme.

## Features

- Real-time cryptocurrency prices from CoinGecko API
- Dark/Light mode toggle with persistence
- Search and filter cryptocurrencies
- Sort by market cap, price, or 24h change
- Detailed coin pages with market statistics
- Responsive design for all screen sizes

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **API**: CoinGecko Public API

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deploy

Deployed on [Vercel](https://vercel.com)
